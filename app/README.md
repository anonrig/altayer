## Altayer Frontend Application

### Dependencies

- Node v6.9.1
- NPM 4.0.5
- Yarn 0.18.1

### Install packages

```javascript
yarn
```

### Run in development mode

(This compiles the files on the go.)

```javascript
yarn start
```

### Run in production mode

(This compiles the code for production.)

```javascript
yarn build
```

### Run on server.

(This compiles and runs the express server.)

```javascript
yarn run server
```
